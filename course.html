Web Application Programming and Hacking

Individual Project: Professional Profile Website

Instructor: Phu Phung

Course Overview

This course introduces the fundamentals of web application development and cybersecurity. Students will begin by building modern front-end applications using HTML5, CSS, and JavaScript, followed by backend development using PHP and MySQL. Through hands-on labs, hackathons, and projects, students will explore critical web vulnerabilities and defense mechanisms, focusing on OWASP Top 10 and secure full-stack development.

Course Timeline & Topics

  • Week 1: Preparation, Lab 0 - Course Setup, Ubuntu VM
  • Week 2: Lab 1 - Foundations of the Web (HTTP, HTML, CSS)
  • Week 3: Lab 2 - Front-End Web Development (Forms, JS)
  • Week 4: Individual Project 1 - Professional GitHub.io Homepage
  • Week 5: Hackathon 1 - XSS Attacks (L8)
  • Week 6: Lab 3 - Secure Web Dev in PHP/MySQL (L9–12)
  • Week 7: Hackathon 2 - SQL Injection (L12)
  • Week 8: Lab 4 - Session Authentication (L14–15)
  • Week 9: Hackathon 3 - Session Hijacking (L16)
  • Week 10–11: Individual Project 2 - Secure Full-Stack Development
  • Week 12–13: Hackathon 4 - CSRF Attacks (L18)
  • Week 13: Final Team Project: Build a miniFacebook Clone

Technology Requirements

To participate in all labs and projects, ensure you have:

  • A laptop or desktop capable of running virtual machines
  • VirtualBox installed
  • An Ubuntu 22.04 Virtual Machine set up
